Instruction
1. Install Visual Studio Code: https://code.visualstudio.com/Download (used version: 1.53.1)
2. Install Node.js: https://nodejs.org/en/ (used version: 14.15.5)
3. Install Firefox: https://www.mozilla.org/pl/firefox/new/ (used version: 85.0.2)
4. Install TestCafe (global): npm install -g testcafe (could be by VS Code terminal)
5. Install TestCafe (local): npm install --save-dev testcafe (all required elements will be placed in node_modules folder; could be by VS Code terminal; set project directory)
6. To run tests, write: npm test (in VS Code terminal)


The tests are based on a page - http://tutorialsninja.com/demo/
The site allows to use it for the purpose of testing
